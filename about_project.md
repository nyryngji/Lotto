# Lotto number prediction project

> **Why?**

- 개인 사심 채우기 및 지적 호기심 

<br>

> **How?** 

- 로또 1등 당첨을 기준으로 데이터 분석을 실행해나갈 예정 
- 데이터 분석의 결과를 토대로 직접 로또 구매 및 결과 공유 예정 
<br>

> **Data source**

- [동행복권](https://www.dhlottery.co.kr/common.do?method=main)
<br>

> **Result**

- 로또 구매 금액보다 당첨 금액이 더 많도록 즉, 손해 보지 않는 것이 목표
